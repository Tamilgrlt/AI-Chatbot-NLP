Project Overview

AI Chatbot with NLP is a simple conversational chatbot developed using the Python programming language and Natural Language Processing (NLP) techniques. The chatbot is designed to understand user queries and provide appropriate responses based on a predefined knowledge base.
The project uses the NLTK (Natural Language Toolkit) library to process user input. NLP techniques such as tokenization and lemmatization are used to analyze the text entered by the user. After processing the input, the chatbot compares it with stored questions in the dataset and returns the most relevant response.
This chatbot can answer basic questions related to technology, programming, and general information. The goal of this project is to demonstrate how Natural Language Processing can be used to build simple AI-based conversational systems.

The chatbot runs in a Python environment and can be executed through VS Code terminal. Users can type questions, and the chatbot responds instantly until the user exits the conversation.
